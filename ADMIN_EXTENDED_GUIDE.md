## 🔄 KẾT NỐI VỚI WEBSITE THẬT

Hiện tại admin đang lưu vào **localStorage**. Để áp dụng lên website thật:

### Bước 1: Tạo API Backend
```javascript
// API endpoints cần tạo
POST /api/content/hero
POST /api/content/features
POST /api/content/pricing
POST /api/content/faq
POST /api/content/menu
POST /api/content/banners
POST /api/content/testimonials
POST /api/content/seo
```

### Bước 2: Thay localStorage bằng API calls
```javascript
// Thay vì:
function saveHomeHero() {
  contentData.homeHero = {...};
  localStorage.setItem('websiteContent', JSON.stringify(contentData));
}

// Dùng:
async function saveHomeHero() {
  const data = {...};
  await fetch('/api/content/hero', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data)
  });
}
```

### Bước 3: Website load từ API
```javascript
// Trong index.html
async function loadContent() {
  const response = await fetch('/api/content/hero');
  const data = await response.json();
  
  document.getElementById('hero-title').textContent = data.title;
  document.getElementById('hero-subtitle').textContent = data.subtitle;
  // ...
}
```

## 🎨 TIPS & TRICKS

### 1. Sử dụng Emoji cho Icons
```
📚 📖 📝 ✨ 💰 🎯 🚀 ⚡ 🔒 💡
👥 📧 📞 🏠 🔗 ⬇️ 🎨 💬 🔍 ✓
```

### 2. Viết mô tả ngắn gọn
- Tối đa 2-3 câu
- Tập trung vào lợi ích
- Dùng từ ngữ đơn giản

### 3. Giá cả
- Luôn hiển thị đơn vị (VNĐ, đ)
- Làm tròn số đẹp (99.000đ thay vì 98.750đ)
- Highlight gói nổi bật

### 4. SEO
- Meta Title: 50-60 ký tự
- Meta Description: 150-160 ký tự
- Keywords: 5-10 từ khóa, cách nhau bằng dấu phẩy

## 🐛 XỬ LÝ LỖI

### Không lưu được?
- Kiểm tra localStorage có bị disable không
- Thử refresh trang
- Xóa cache browser

### Mất dữ liệu?
- Dữ liệu lưu trong localStorage
- Nếu xóa cache browser → mất dữ liệu
- **Khuyến nghị:** Export thường xuyên

### Preview không cập nhật?
- Kiểm tra đã nhập đúng field chưa
- Refresh trang và thử lại
